## 项目说明

本项目的初衷是为大家提供一个方便查阅、社区共建、真实有温度的Web3&AI面试题集合。

项目目录按照 **岗位类别 -> 真实面经 + 技术栈 + 业务场景** 的结构设置，说明如下：

岗位类别

- 真实面经
- 技术栈：专业技术（内容为**一问一答**）
- 业务场景：岗位工作中会遇到的实际业务场景，比如合约部署、钱包集成、项目设计等等（内容为**一问一答**）
